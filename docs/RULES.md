# Magiclike — Comprehensive Rules

> **Moved.** The rulebook is now an interlinked cluster under [`docs/wiki/rules/`](wiki/rules/) — one page per `§`, co-located with the durable-concepts wiki and mounted into the Obsidian vault. **This stub is a permanent redirect** so existing `RULES.md §NNN` citations keep resolving; the canon lives in the pages below. Start at the [rulebook index](wiki/rules/README.md).

**Canon:** how the game works, in plain English, independent of any implementation. When the doc and the code disagree, **the doc wins** — the code is the patient. Each section page ends with an **Implementation status** note flagging where the Godot port and html-proto deviate; the live cross-engine tracker is [`DIVERGENCE.md`](DIVERGENCE.md).
