# Notes on Forest Titan

Type changed June 2026: `Giant` → `Giant Treefolk`.

The "Forest Titan" name reads as a primordial forest being, not a humanoid giant.
Making it a Giant Treefolk gives it both a unique identity (one of very few Giant-typed cards)
and tribal synergy with the Treefolk lord (Ancient Treant).

**Art needs a redo.** The current art depicts a humanoid giant, not a treefolk.
When regenerating: the creature should read as a massive ancient tree given physical form —
bark-skin, root-limbs, canopy crown. The "only source of light is X" technique applies
if you want the forest atmosphere to land (dappled undergrowth light, bioluminescent fungi).
Stat targets: 5/5 trample feels right for the top end of the Treefolk tribe.
